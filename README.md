# HobbyGameEngine
This is a game engine in created in cpp using graphics.h library.

## GameEngine Features: 

1. Entity Component System Architecture
2. Event Systems
3. Rendering systems with "graphics.h" library

## Development Env:

1. CLion v2021.3.3
2. CMake v3.21.1
3. Win32API
4. MinGW-w64 64-bits (gcc v11.2.0)
